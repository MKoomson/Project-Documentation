**Operational Expense Tracker**

## Overview
This project is a Google Sheets-based operational expense tracking system designed to monitor daily spending, categorize expenses, and provide simple financial insights through an interactive dashboard.

---

## Objective
To build a simple, structured, and efficient system for tracking and analyzing expenses in a way that supports better financial decision-making.

---

## Features
- Daily expense logging (date, item, category, amount, payment method)
- Category-based expense tracking using dropdown lists
- Automated total expense calculation
- Expense breakdown by category using built-in formulas
- Monthly expense summary
- Visual dashboard with charts for quick insights

---

## Tools Used
- Google Sheets
- Data Validation (Dropdown lists)
- Functions: SUM, QUERY
- Charts (Pie chart for category distribution)

---

## Key Highlights
- Designed a structured data entry system to reduce errors
- Implemented automated calculations for real-time tracking
- Built a summary dashboard for financial visibility
- Applied basic data analysis using QUERY function
